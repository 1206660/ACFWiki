# SlimWiki Local Mirror

## Usage

```bash
npm install
npm run mirror
npm run serve
```

One command startup:

```bash
npm run bootstrap
```

## Config

Optional environment variables:

```bash
SLIMWIKI_START_URL=https://slimwiki.com/dark-tower-int/acfu/welcome
SLIMWIKI_PAGE_PREFIX=/dark-tower-int/acfu
HOST=0.0.0.0
PORT=80
```

## Output

- Rendered HTML: `output/site`
- PDFs: `output/pdf`
